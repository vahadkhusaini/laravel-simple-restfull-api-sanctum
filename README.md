## How to Use This Project

I recommended to use Postman for testing this Rest Server

SET UP ENV
- clone this repo/download
- open to your favorite text editor, visual code is good :D
- edit .env according to your database settings
- run php artisan migrate
- run php artisan db:seed --class=UserTableSeeder
- run php artisan serve

TESTING FOR POSTMAN
- Open Postman
- Click import file, choose folder postman_testing
- then click Laravel sanctum test.postman_collection.json 
- Import and Enjoy!
